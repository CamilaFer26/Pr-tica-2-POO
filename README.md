## Pr-tica-2-POO
# Atividade Prática 2
Atividade avaliativa da disciplina de Programção Orientada a Objetos, que propõe a implementação de um sistema de controle de estoque simples, com cadastro de produtos, consulta de itens, entrada e saída de 
estoque e relatórios simples. A implementação utiliza abstração, herança, polimorfismo, sobrescrita, sobrecarga, final, static, HashMap e List.
